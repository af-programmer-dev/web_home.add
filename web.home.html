Men haqimda:
Men Abdulvahhob, 14 yoshli frontend dasturchiman. Web dasturlashga qiziqaman va o‘z loyihalarim ustida ishlashni yoqtiraman. Hozir shaxsiy portfoliyo saytimni yaratmoqdaman, unda profil tizimi, real-time chat va foydalanuvchilar uchun ro‘yxatdan o‘tish imkoniyati bo‘ladi.

Saytimning frontend qismini HTML, CSS va JavaScript yordamida ishlab chiqyapman, backend esa Node.js va MongoDB yoki Firebase bilan amalga oshiriladi. Maqsadim – zamonaviy va interaktiv web-loyihalar yaratish hamda tajribamni oshirish.



Bu sayt – mening shaxsiy portfoliyo loyiham. Sayt orqali men o‘z ishlarimni taqdim etaman, yangi loyihalarimni qo‘shaman va foydalanuvchilar bilan muloqot qilish imkoniyatini yarataman. Saytda quyidagi imkoniyatlar mavjud:

✅ Profil tizimi – Foydalanuvchilar ro‘yxatdan o‘tib, tizimga kirishlari mumkin.
✅ Portfoliyam – Ishlagan loyihalarim va kelajakdagi rejalarim haqida ma’lumotlar.
✅ Real-time chat – Saytga tashrif buyurgan foydalanuvchilar o‘zaro muloqot qilishlari uchun.
✅ Admin panel – Yangi ma’lumotlar va loyihalarni qo‘shish imkoniyati.

Sayt frontend qismi HTML, CSS va JavaScript yordamida yaratilgan. Backend esa Node.js va MongoDB yoki Firebase orqali amalga oshiriladi. Saytni rivojlantirib borish va yangi funksiyalar qo‘shish ustida doimiy ishlayman.
